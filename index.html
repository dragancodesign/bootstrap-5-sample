Tables in Bootstrap 5


Table Cells

Each table cell is defined by an opening "td" and an "/td" closing tag.

Here I have also used mt-4: Margin Top class:

mt-0 = 0

mt-1 = 0.25

mt-2 = 0.5

mt-3 = 1.0 => This is default !

mt-4 = 1.5

mt-5 = 3.0

"th" stands for Table Headers.
"tr" stands for Table Row.
"td" stands for Table Data.

We have to use table classes (example): class="table table-striped table-hover" +
here I have also used class= "mt-4" to make margin from the top of the table element +
class="ps-3" is Padding Start class="pe-2" Padding End +
Margin-Top on Large screen, margin-top on medium screen & margin-top on small screen class="mt-lg-0 mt-md-3 mt-sm-0".
class="pb-4" means Padding Bottom.
